HTML

HTML is code used to structure your webpage. The way it is structured is by using things called "Elements".

But what are elements?

Elements are keywords or letters used to help define how you want text or images to be structured on your webpage.

What are some of these HTML elements?

Types of HTML elements

A quick list of elements commonly used and what they are used for

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element is used to define different characteristics used for a webpage, such as charset, which sets the types of characters that the webpage will be using.
  • The p element is used to create a line of text without any special function
  • The h1,h2,h3,h4,h5 and h6 elements are used to structure your webpage using differently sized and bolded text. You should only ever have 1 h1 element on a webpage as that is to be used to introduce your page.
