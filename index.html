HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element is used to define different characteristics used for a webpage, such as charset, which sets the types of characters that the webpage will be using.
  • The p element is used to create a line of text without any special function
  • The h1,h2,h3,h4,h5 and h6 elements are used to structure your webpage using differently sized and bolded text. You should only ever have 1 h1 element on a webpage as that is to be used to introduce your page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The keyword "Class" is used to define specific attributes to a more individual and distinct area in your code.
  • You can make all elements of a specific type similar using CSS. For example, changing all elements of h1 to be a specific color or size.
